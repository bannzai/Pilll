# settings QA

設定画面は項目数が多いため、主要フローを中心にテスト項目を定義する。

## 1. アカウント

- [x] **連携状態の表示**: 「アカウント設定」行はApple/Googleいずれも未連携なら「未登録」、いずれか連携済みなら「連携済み」と表示される
- [x] **アカウント連携・再ログイン導線**: 「アカウント設定」から連携一覧画面に遷移し、未連携のApple/Googleで「連携する」をタップするとサインインシートが表示される。連携済みの行を長押しすると「認証情報を更新します」の再ログイン確認ダイアログが表示される。実OAuth認証結果の確認はSimulatorでは制限があるため、ダイアログ・シートの表示までを確認対象とする
- [x] **退会**: 連携一覧画面下部の「退会する」をタップすると確認ダイアログが表示される。退会は不可逆操作のため、実際の削除実行はQA専用の使い捨てアカウントでのみ行い、通常は確認ダイアログの表示までに留める
