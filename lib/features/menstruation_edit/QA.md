# menstruation_edit QA

## 1. 新規作成（日付範囲ピッカー）

- [ ] **新規作成ピッカー表示**: 生理タブの記録フローから新規作成用のピッカーを開くと、ヘルプテキストに「生理開始日を選択」が表示され、開始日・終了日を選択できる
- [ ] **選択可能な日付範囲**: 日付範囲ピッカーで選択できるのは2020年1月1日から今日+30日までで、それ以外の日付は選択できない
- [ ] **保存で新規記録が作成される**: 開始日・終了日を選んで確定すると新しい生理記録が作成され、「◯月◯日から生理開始で記録しました」のスナックバーが表示される
