## 2. 服用記録

- [ ] **服用ボタンで記録**: 画面下部の「飲んだ」ボタンをタップすると本日分の錠に服用済みマークが付き、ボタンが「飲んでない」（取り消し）ボタンに切り替わる
- [ ] **シート上マーク直接タップで服用/取消**: ピルシート上の本日分でまだ未服用の錠マークを直接タップすると服用記録が付く。全て服用済みの状態でシート上の服用済みマーク、または「飲んでない」ボタンをタップすると直前の服用記録が取り消され「飲んだ」ボタンに戻る
- [ ] **未来の錠はタップ無効**: 本日の錠番号より先（未来）の錠マークをタップしても服用記録は付かず、画面表示・ボタン状態は変化しない

## 4. 休薬期間管理

- [ ] **休薬開始**: ピルシート設定シートの「休薬を始める」をタップすると確認ダイアログ（服用番号は進まない旨の注意書き付き）が表示され、開始を選ぶと休薬が始まりホーム下部のボタンが操作不可の「休薬中」ボタンに変わる
- [ ] **服用済み時は休薬開始不可**: 本日分を服用済みの状態で「休薬を始める」をタップすると、休薬前に本日の服用を取り消すよう案内するエラーダイアログが表示され、休薬は開始されない
- [ ] **休薬終了と期間変更**: 休薬中にピルシート設定シートの「服用を再開する」をタップすると休薬が終了し完了のスナックバーが表示されて通常の「飲んだ」ボタンに戻る。終了後は設定シートの休薬期間一覧から対象期間をタップし、日付ピッカー（単日または期間）で開始日・終了日を変更できる
