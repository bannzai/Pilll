## 5. エッジケース（上限・下限）

- [x] **上限到達時は追加ボタン非表示**: 通知時間が3件（上限）登録されている状態では、一覧最下部の「通知時間の追加」ボタン自体が表示されない

## 6. タイムゾーン設定

- [x] **同期ダイアログの表示**: 画面右上の時計アイコンをタップすると「端末のタイムゾーン(◯◯)と同期しますか？」という確認ダイアログが、現在設定されているタイムゾーン名とともに表示される
- [ ] **「はい」で同期しスナックバー表示**: ダイアログで「はい」をタップするとタイムゾーンが端末のものに変更されてダイアログが閉じ、画面下部に「〜に変更しました」というスナックバーが表示される
- [x] **「いいえ」でダイアログのみ閉じる**: ダイアログで「いいえ」をタップするとダイアログが閉じるだけで、タイムゾーン設定は変更されない
