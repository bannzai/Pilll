# diary_post QA

## 1. 体調・体調詳細の記録

- [ ] **体調ボタンのトグル**: 「体調」欄の怒り顔（悪い）／笑顔（良い）アイコンをタップすると選択状態になり背景色が変わる。選択中のアイコンを再度タップすると選択解除される。片方を選択すると他方の選択は自動的に解除される
- [ ] **体調詳細チップの複数選択（無料ユーザー）**: 無料ユーザーは頭痛・腹痛・吐き気など既定の症状チップ一覧から複数タップで選択・解除ができる
- [ ] **体調詳細の編集導線分岐**: 「体調詳細」欄の鉛筆アイコンをタップすると、プレミアム/トライアル中ユーザーは症状項目編集シート（diary_setting_physical_condtion_detail）が開き、無料ユーザーはプレミアム紹介シートが表示される
  - Note: プレミアム版の項目編集確認にはプレミアム/トライアル状態のテストアカウントが必要
