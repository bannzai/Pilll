# diary_post QA

## 1. 体調・体調詳細の記録

- [x] **体調ボタンのトグル**: 「体調」欄の怒り顔（悪い）／笑顔（良い）アイコンをタップすると選択状態になり背景色が変わる。選択中のアイコンを再度タップすると選択解除される。片方を選択すると他方の選択は自動的に解除される
- [x] **体調詳細チップの複数選択（無料ユーザー）**: 無料ユーザーは頭痛・腹痛・吐き気など既定の症状チップ一覧から複数タップで選択・解除ができる
- [x] **体調詳細の編集導線分岐**: 「体調詳細」欄の鉛筆アイコンをタップすると、プレミアム/トライアル中ユーザーは症状項目編集シート（diary_setting_physical_condtion_detail）が開き、無料ユーザーはプレミアム紹介シートが表示される
  - Note: プレミアム版の項目編集確認にはプレミアム/トライアル状態のテストアカウントが必要
  - ⏭️ スキップ（一部）: 無料ユーザー分岐（プレミアム紹介シート表示）は未検証。dev環境の新規匿名アカウントは RemoteConfig の `trialDeadlineDateOffsetDay`（45日）によりアカウント作成直後から自動的にトライアル状態になるため、鉛筆アイコンをタップすると症状項目編集シートが開くトライアル分岐が発生した。無料ユーザー分岐の確認には非トライアル（トライアル期限切れ）のテストアカウントが必要

## 4. 保存・キャンセル

- [x] **日記の保存**: 体調・体調詳細・セックス・メモを入力後に右上「保存」をタップすると日記が保存され画面が閉じる。カレンダーに戻ると入力内容に応じたアイコンが該当日に表示される
- [x] **未保存で閉じる**: 左上の×ボタンをタップすると保存確認なしで画面が閉じ、入力内容は保存されない

## 5. 日記確認シート（登録済み日記の表示）

- [x] **確認シートの表示内容**: 日記が登録済みの日をタップすると確認シートが表示され、日付・体調アイコン・体調詳細チップ・セックスアイコン（記録がある場合）・メモが表示される
- [x] **編集導線**: 確認シートの鉛筆アイコンをタップすると当該日記の内容が反映された日記投稿画面（編集モード）に遷移する
- [x] **削除導線**: 確認シートのゴミ箱アイコンをタップすると削除確認ダイアログが表示され、「削除する」で日記が削除されカレンダー画面に戻る。「キャンセル」でダイアログのみ閉じ日記は保持される
